# Veiculos Monitor

Este projeto é um exemplo de aplicação Java usando Maven, JPA e Hibernate.

## Entidades
- Caminhao
- Carro
- Moto
- Proprietario
- Veiculo (classe base)

## Tecnologias
- Maven
- JPA/Hibernate
- H2 Database

## Como executar
1. Clone o repositório.
2. Compile e execute o arquivo `Main.java`.
3. Veja a saída no console ou no GUI.
